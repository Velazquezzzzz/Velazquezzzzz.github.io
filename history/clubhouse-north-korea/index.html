你怎么没有记下来啊

今天 Clubhouse 的一个 Room 里听了一下午十分有料的八卦➕后面有人提到了 drugs 的体验以及 burning man 之类的一些奇奇怪怪新奇的东西

# 电子产品

朝鲜仅对外国人开放 3g/4g 用的阿里郎牌📱
网费挺贵的 当地人商店和外国人商店（以美元 / 人民币结算）

# 金日成学习堂

阅兵的时候站的地方，不能摆和某三胖一样的动作

# 杂

可以去朝鲜见亲戚（不能带 English/Korean）

两个导游带一个 18 个人的团 （高端阶层生活和我们差不多 不能换朝鲜币 （导游 A 给的几千块朝鲜币是导游 B 一个月的工资

60 年前在朝鲜打仗，60 年后回来看，人倒在谈判桌，死在谈判桌上（共同警备区太长

封国之前，金日成广场看晚会，大家都是站在广场上 10w 人左右，外宾团走到最前面，2-3 小时的歌舞表演，牡丹峰压轴，最后倒数过年放烟花，全国直播的跨年晚会

住的都是羊角岛酒店
羊角岛酒店地下有赌场

阿里郎（十万人团体舞，对面小学生不停的换图片，机械整齐，08 的时候平壤热水使用受限，冬天可能 2-3 个月才能洗澡，当时拍照不能随便拍路上的行人，在火车上被抓了，一车厢的人被查照片，花了五千块行贿了查照片的人，很怕在北朝鲜被枪毙了）

朝鲜有名创优品，17 左右开的，后来名创优品去美国上市，所以就把朝鲜的控股交出去了

鸭绿江上拍照，给十几块钱就能接受了

# 政治

泰永浩，脱北者，保守党，首尔当了议员
疫情期间，泰永好说金正恩病危，后面自己名誉受损

# 朝鲜冷面

平壤冷面（清汤 —- 最正宗的
深度游玩的方向

  • 平壤科学技术大学（招汉语 / 英语老师 和美国合办的
  • 金日成综合大学（朝鲜语语言课，纠正南朝鲜的口音
  • 大同江大桥不坚固，导游会跟踪你，可以甩掉，一个照相机才 30 多块钱 rmb

# 两个导游

沪普老金，72 道行政区划（算了韩国的），后来说只有 36 道

# 小金

聊了价值观，家庭观，婚龄，小金会看韩剧，看到三角恋之类的情节，不知道好不好，朝鲜平均女生婚龄 30 岁左右，对中文资源接触较多，每晚脱队，晚上观察到的景象：平壤的街道灯火通明，两旁住宅楼也是灯火通明，不拉窗帘，可以看到家具，领导人画像和大立柜

日间脱队行动：去大街上转悠，拍了普通人肖像，被人举报，导游训斥了一番，最后一天早上九点左右，当时 12 年不能带手机，军事管制区和有守卫的地方不能照相，最后迟到了差点回不去了，幸好同学拖住了

# XING 西北一个大叔，1995 年去的朝鲜

十岁左右中朝少年沟通夏令营去交流，金日成刚去世，所以延迟了两周（国丧），丹东呆了两周，去了黑市，买了朝鲜货币，1rmb 买 100 朝鲜币

当时学了一首金日成将军之歌，当时平壤比北京繁华的多，地铁干净，高楼很高，每次上完菜（精美，量少），服务员就消失。去酒店二楼买麦芽糖来果腹。一起玩的小学生他们都很快乐，最深印象的是招待在国际儿童村（外面一个金日成雕像，游乐场完全没有人，免费绿豆汤喝，去娱乐室玩红白机）那个酒店现在还没建完，差一个尖的金字塔。

因为金日成刚去世，所以每天只能看两小时电视，有个人会走出来，讲金日成的故事，然后痛哭流涕，然后雪花。

# 一个女生，爷爷出生在北朝鲜半岛

90 年代的一个故事
后面留学到中国
每年会送物资给北朝鲜的亲戚
送物资的方式

  • 写信，比如说时间地点在哪里等
  • 亲戚来自己家，拿很多东西，再回北朝鲜
    遇到脱北者
  • 父母都不在了
  • 晚上游过鸭绿江
  • 在树林里躲了三天被人发现
  • 有她家的联系方式
  • 20 多岁的成年女孩，照顾了一个月左右，怕被抓走，送到青岛，饭店打工，几年后据说在餐厅干了不到半年，去 ktv 做了小姐，说是来钱比较快

# Lee 谈脱北者

本来逃到中国是应该送回去的，但是送回去后就枪毙了，后面就是送韩国大使馆，自行处理

建国后，河南新乡，专门朝鲜脱北者聚集地，从河南去港澳，再去了韩国

# Fred

自建团➕导游，18 年去朝鲜元山市，能看到很多贫瘠山村，路也不太能走。能看到征兵，能看到海边有上下学的孩子。
大楼

  • 一年内建完（变成三年建完）
  • 涂料定期有人搞
  • 所有建筑都没怎么装空调
  • 住在高丽饭店，特级宾馆，顶上有旋转餐厅，有一半的房间是被挡住的（可能是什么军事用处），朝鲜导游有手机的（和 iPhone 很像，只能用 2G）
  • 夜晚，当地游乐场从早上玩到晚上
  • 朝鲜的地方特色，分房制
  • 有梦想，就写下心愿，就能选上读书（然而说不出来）
  • 对领导人的称呼，历代都不同，永远的 xxx
  • 孩子玩的，以上海为背景，打日本人的游戏

# 一个男生

鸭绿江

  • 🇨🇳这边的公路很好，村也发展不错，朝鲜那边都是小房子，无法判断有没有住
  • 沿途有军事设施
  • 中间有条火车线，有运输东西
  • 还有小型火车站，之前每个站上面都有金日成头像
  • 鸭绿江很窄
  • 一般情况下治安挺好，但有时候晚上会有人过来偷东西，偷日常用品，我们这是村里登记，然后有补偿
  • 临江（共同城市，县），他们那是村
  • 往那边扔饮料，有捡走的，晚上就黑了，只有路灯，偶尔有手电巡逻的

# zhongda

跟了一个画廊团（家具城，面向富人

  • 丹东到平壤的火车上，遇上一个美国韩裔平壤科技大学行政级工作人员（后来回国后，有人因为间谍行为被抓了，不知道有没有这个人
  • 八月份类似于建军节，放假，就去海边游玩，
  • 拍照与政治气候有关，可能宽松的时候，就随便拍，但漏洞是 sd 卡，或者是删掉的照片，可以恢复备份
  • 签证是在纸上，留下出入境的痕迹，后来去美国，不能隐瞒这个事情
  • 北京出发去平壤，就看不出来

现在去平壤，有市民已经开始养宠物了
